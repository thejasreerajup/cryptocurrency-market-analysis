#  Cryptocurrency Market Analysis & Prediction

## Project Overview

This project aims to analyze cryptocurrency market trends and predict Bitcoin prices using Machine Learning and Deep Learning techniques. It focuses on extracting meaningful insights from historical data and building predictive models to forecast future price movements.

---

##  Key Features

* 🔹 Data preprocessing and cleaning
* 🔹 Exploratory Data Analysis (EDA)
* 🔹 Visualization of cryptocurrency trends
* 🔹 Implementation of Machine Learning models
* 🔹 Deep Learning model training and evaluation
* 🔹 Bitcoin price prediction and trend forecasting

---

## Technologies Used

* **Programming Language:** Python
* **Libraries & Tools:**

  * NumPy
  * Pandas
  * Matplotlib
  * Scikit-learn
  * TensorFlow / Keras

---

## How to Run

 Launch Jupyter Notebook 
* Open the following notebooks:
 1. Cryptocurrency_Market_Analysis.ipynb 
 2. bitcoin_model_training.ipynb .
* Run the cells sequentially
